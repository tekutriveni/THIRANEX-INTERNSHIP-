Design Rationale

Key Design Decisions

Every decision made with purpose — backed by UX principles and user needs.

📐
Low-fidelity wireframes chosen
Quickly validate user flows before high-fi investment. Allows fast iteration based on feedback.
📱
390×844 frame size
iPhone 14 Pro standard; scales well to most Android screens. Industry standard for mobile wireframing.
🧭
4-tab bottom navigation
Home, Search, Cart, Profile covers 90% of user actions. Reduces cognitive load significantly.
🛒
Persistent cart indicator
Users can always see items in cart without interrupting browsing flow.
⚡
3-step progressive checkout
Address → Payment → Review reduces abandonment vs single long form.
📍
Real-time tracking screen
Key differentiator. Map + driver info + progress steps reduces post-order anxiety.
Design Principles

Guiding Principles

Four principles that shaped every design decision in FoodRush.

01
Speed First
Every screen optimized for under 3 taps to core action. Checkout in 3 steps max.
02
Clear Hierarchy
Primary action always in orange CTA. Secondary actions in neutral tones.
03
Trust Signals
Ratings, times, and driver info always visible to reduce ordering anxiety.
04
Minimal Friction
Progressive disclosure: show only what's needed at each step. No form overload.
