Usability Audit

Heuristic Evaluation

8 usability issues identified using Nielsen's 10 Usability Heuristics on existing food delivery interfaces.

# Heuristic Issue Found Severity Redesign Fix
1Visibility of System StatusNo loading indicators or order progress feedbackHigh✓ Added progress bar and live status badges
2Match Between System & Real WorldTechnical labels like "SKU" confuse everyday usersMedium✓ Replaced with plain language throughout
3User Control & FreedomNo easy way to remove or edit cart itemsHigh✓ Added inline edit, delete, and undo
4Consistency & StandardsInconsistent button sizes and styles across pagesMedium✓ Unified button component system
5Error PreventionNo order review step before final paymentHigh✓ Added mandatory order review screen
6Recognition over RecallMenu categories disappear on scrollMedium✓ Sticky category tabs always visible
7Flexibility & EfficiencyNo quick reorder option for returning usersLow✓ "Order Again" section on home screen
8Aesthetic & Minimalist DesignCluttered home screen with excessive bannersMedium✓ Reduced banners, added whitespace
